

### Prerequisite


It requires Python 3.6 and [Graphviz](https://www.graphviz.org/)

### Commands
```
# using pip (pip3)
$ pip install diagrams

# using pipenv
$ pipenv install diagrams

# using poetry
$ poetry add diagrams
```
### Docs
